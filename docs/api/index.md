# API Reference

This section documents the four core Perl analysis scripts used by the **codonanalyzer** pipeline.

| Script | Description |
|--------|-------------|
| [codon.pl](codon.pl.md) | Compute codon usage, nucleotide composition, RSCU, ENc, ORF statistics, and 3-mer Shannon entropy from a DNA FASTA record. |
| [longORF.pl](longORF.pl.md) | Identify the longest open reading frames (ORFs) in all six translation frames of a FASTA sequence. |
| [translate.pl](translate.pl.md) | Translate DNA FASTA sequences into protein sequences using the standard genetic code. |
| [hydropathy.pl](hydropathy.pl.md) | Compute per-residue and sliding-window Kyte–Doolittle hydropathy scores for translated protein sequences. |
